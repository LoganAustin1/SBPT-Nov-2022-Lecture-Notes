HTML: HighperText markup language.

An HTML document is composed of elements and attributes

Element: Building blocks of HTML that is also a node. Part of the HTML document that tells a web browser how to structure and interpret a part of the HTML document. HTML elements can contain formatting instructions, semantic meaning, and content.

CSS Definitions:

CSS: Cascading Style Sheets

EMS: font size

Selectors: defines which part of the html element you want to add a ruleset too. CSS begins with a selector, followed by curly braces. Declare your styles inside the curly braces.


examples of selectors include: Class which can be selected with a .
id which can be selected with a #

Ruletset: A CSS rule set contains one or more selectors and one or more declarations. The selector(s), which in this example is h1, points to an HTML element. The declaration(s), which in this example are color: blue and text-align: center style the element with a property and value. The rule set is the main building block of a CSS sheet.
h1 {
color: blue;
text-align: center;
}
